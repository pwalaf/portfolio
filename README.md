# Portfolio — Parfait Rakotomalala

Personal portfolio built with vanilla HTML, CSS and JavaScript.

## Stack

- HTML / CSS / JavaScript — no framework
- Cloudinary — video hosting
- Vercel — deployment

## Features

- Dark / light theme
- FR / EN language toggle
- Offline-first lazy video loading
- Burger menu on mobile
- Scroll animations

## Local development

Clone the repo and open `index.html` in your browser — no build step required.
